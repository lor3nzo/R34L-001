# R34L-001: CSV Validator

A static, in-browser CSV validator.

Validates:
- Well-formed CSV parsing (quotes, escaped quotes)
- Consistent column counts
- Optional type checks (free: up to 10 checks, validates first 100 data rows)

Exports:
- Detailed `errors.csv` with `raw_line` (exact original line)

Deployed at: https://001.r34l.com
